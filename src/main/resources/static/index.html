Documentação do Front-End

Esta seção detalha as funcionalidades e menus da interface do usuário da aplicação.

Menus Principais

  • Dashboard

    Visão geral das principais métricas e informações da aplicação.

  • Animais

    Gerenciamento de dados relacionados aos animais.

    • Pesquisar: Localizar animais cadastrados.
    • Cadastrar: Adicionar novos animais.
    • Editar: Alterar informações de animais existentes.
  • Clientes

    Gerenciamento de dados relacionados aos clientes.

    • Pesquisar: Localizar clientes cadastrados.
    • Cadastrar: Adicionar novos clientes.
  • Finanças

    Controle financeiro e relatórios.

    • Vendas: Visualizar e gerenciar registros de vendas.
    • Relatórios: Gerar relatórios financeiros.
  • Configurações

    Configuração de preferências e gestão de usuários.

    • Conta: Gerenciar informações da conta do usuário.
    • Alterar Senha: Atualizar a senha de acesso.
    • Usuários: Gerenciar usuários do sistema.
    • Sair: Encerrar sessão no sistema.
